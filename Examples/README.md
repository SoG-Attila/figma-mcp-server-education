# Example Materials

This folder contains example resources to help students and instructors get the most out of the Figma MCP Server.

## Files

### prompts.md
30+ ready-to-use Claude prompts for:
- Content calendar generation
- Email campaign creation
- Social media posts
- Landing page copy
- Strategy analysis
- Creative briefs

Students can copy these prompts and adapt them to their specific Figma files.

## Usage

1. Complete the main setup tutorial first
2. Ensure your MCP server is running
3. Open Claude Desktop
4. Copy a prompt from `prompts.md`
5. Adapt it to your specific Figma file structure
6. Experiment and iterate

## Tips

- Start with simple extraction prompts
- Build up to complex content generation
- Save prompts that work well for you
- Share successful prompts with classmates
