Once your Figma MCP server is running, here are effective prompts to get the most out of Claude AI.

## üìÖ Content Calendar Generation

### 2-Week Social Media Calendar
```
Based on the persona in section_1 of my Figma file, 
create a 2-week content calendar for Instagram. 
Include:
- Post topic
- Caption draft
- Best posting time
- Hashtag suggestions
- Content format (carousel, reel, static)
```

### Monthly Blog Calendar
```
Using the customer journey map in my Figma file, 
create a monthly blog content calendar that addresses 
each stage of the journey. Include:
- Week
- Blog title
- Target keywords
- Journey stage addressed
- CTA recommendation
```

## üåê Website & Landing Page Copy

### Landing Page
```

Figma file, write complete copy for a landing page including:
- Hero headline & subheadline
- Above-fold CTA
- 3 benefit sections
- Social proof section
- FAQ section
- Final CTA

### Product Page
```
Using the product features in my Figma file, create an

- Compelling product description

- Use case scenarios
- Technical specifications
- Pricing justification

## üì± Social Media Content

### LinkedIn Posts
```
Based on the thought leadership themes in my Figma file, 
write 5 LinkedIn posts (each 1,300 characters max) that 
position me as an expert. Include:
- Hook
- Value/insight
- Personal story or example
- Call to engage
```

### Instagram Captions
```
Create 10 Instagram caption variations based on the brand 
voice guidelines in section_3 of my Figma file. Mix of:
- Educational posts (3)
- Inspirational posts (3)
- Behind-the-scenes posts (2)
- Product showcase posts (2)

Include emoji, line breaks, and hashtag recommendations.
```

## üìä Strategy & Analysis

### Content Strategy Audit
```
Review all the content from my Figma file and:
1. Identify gaps in the messaging
2. Suggest improvements for clarity
3. Recommend additional content pillars
4. Assess alignment with target persona
```

### Competitive Positioning
```
Based on my unique value proposition in the Figma file, 
write a competitive positioning statement and suggest 
3 ways to differentiate our messaging from competitors.
```

### Brand Voice Analysis
```
Analyze the brand voice examples in my Figma file and:
- Define 3-5 voice attributes
- Create a voice & tone guide
- Provide do's and don'ts
- Write 5 example sentences in this voice
```

## üí° Advanced Prompts

### Persona-Based Content Matrix
```
Extract all personas from my Figma file and create a 
content matrix showing:
- Persona name
- Top 3 pain points
- Content topics that address each pain point
- Recommended content formats
- Distribution channels

Present as a table.
```

### Customer Journey Content Mapping
```
Map the customer journey stages from my Figma file to 
content recommendations:
- Awareness stage: 3 content ideas
- Consideration stage: 3 content ideas  
- Decision stage: 3 content ideas
- Retention stage: 3 content ideas

For each, specify format, distribution, and goal.
```

### Brand Messaging Framework Export
```
Extract the messaging framework from my Figma file and 
format it as:
1. Core message (one sentence)
2. Supporting pillars (3-5)
3. Proof points for each pillar
4. Audience-specific variations
5. Elevator pitch (30 sec)
6. Extended pitch (2 min)
```
