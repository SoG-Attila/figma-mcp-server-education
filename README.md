# Documentation

This folder contains the complete tutorial for students to set up and use the Figma MCP Server.

## Main Tutorial

**[student-tutorial.html](student-tutorial.html)** - Complete step-by-step guide (~90 minutes)

Open this file in your web browser for:
- Python installation instructions (Windows & Mac)
- Project setup and configuration
- Figma API token generation
- Claude Desktop configuration
- Testing and troubleshooting
- Next steps and example prompts

## How to Use

### For Students
1. Download or clone this repository
2. Open `student-tutorial.html` in any web browser
3. Follow every step carefully
4. Ask for help if you get stuck
5. Celebrate when it works!

### For Instructors
1. Review the tutorial before class
2. Test on both Windows and Mac if possible
3. Prepare to help with common issues
4. See [TEACHING-GUIDE.md](../TEACHING-GUIDE.md) for lesson plans

## Prerequisites

Before starting the tutorial, ensure you have:
- A computer with admin rights
- Windows 10/11 or macOS 10.15+
- A Figma account (free)
- A Claude account (free)
- A Figma file with content to extract

## Time Required

- **First-time setup:** 90 minutes
- **With instructor help:** ~2 hours (includes breaks)
- **Daily use:** Just open Claude Desktop (automatic)
